# news: новости для аудитора

## Порядок (быстро → глубже)
1. Свежий пул дня УЖЕ собран в БД (≈40 новостей за 48ч: ЦБ, banki.ru, Frank Media,
   Коммерсантъ, РБК, ТГ-каналы инцидентов/схем мошенничества):
     alsql "SELECT payload FROM daily_digest WHERE section='news' ORDER BY digest_date DESC LIMIT 1"
   В payload: groups[] — редакторские группы с summary/severity; pool[] — сырые
   title/url/source/ts/snippet. Сначала ищи ответ здесь — это мгновенно.
2. Мало → живой веб-поиск (внутренний SearXNG, внешние поисковики через него):
     curl -s "http://127.0.0.1:8888/search?q=ЗАПРОС&format=json" | head -c 4000
3. Нужен текст конкретной страницы — скачай и прочитай сам (curl + разбор).

## Формат ответа
Каждая новость: суть 1-2 предложения + дата + [источник](url) инлайн.
Отбирай Сбер-релевантное; конкуренты — только как рыночный контекст.
