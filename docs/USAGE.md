# Как пользоваться AuditLens

AuditLens — это аналитический помощник для аудиторов банковского сектора. Задаёшь вопрос на русском — получаешь структурированный отчёт со ссылками на первоисточники, графиками и PDF-экспортом.

---

## Два режима работы

### 1. Quick Mode — быстрый ответ (без deep-research)

Простые вопросы (5-15 сек): «какие у Сбера ставки по семейной ипотеке», «топ-5 жалоб клиентов ВТБ за месяц».

Просто введи в чат и нажми **Send**. Чек-бокс `Deep Research` не включай.

### 2. Deep Research — глубокое исследование (1-3 мин)

Сложные сравнительные вопросы. Включи чек-бокс **🔬 Deep Research** перед отправкой.

Pipeline сделает:
1. **Resolver** — поймёт тему, банки, аудиторию
2. **Planner** — разложит на 10-16 шагов
3. **Сбор данных** — semantic search в БД + real-time fetch с сайтов банков
4. **Fact extraction** — на каждый банк извлечёт ключевые цифры
5. **Claim verification** — отфильтрует факты которых нет в источниках (анти-галлюцинация)
6. **Synth + critic + agent-loop** — несколько LLM-проходов: draft → critic → 2 итерации добавления
7. **Merge** — финальная редактура в один чистый markdown
8. **Charts** — автоматическая визуализация числовых сравнений
9. **PDF-экспорт** — кнопка `Download PDF`

## Примеры хороших вопросов

### Сравнительные (deep mode даёт максимум)

| Категория | Пример |
|---|---|
| **Карты** | `Сравни условия премиальных дебетовых карт Сбер Прайм, Тинькофф Premium, Альфа Wealth по комиссиям, кешбэку, привилегиям` |
| **Ипотека** | `Семейная ипотека: ставки, первый взнос, требования в Сбер, ВТБ, Альфа-банк, ДомРФ` |
| **Вклады** | `Сравни валютные вклады в долларах и юанях в топ-10 банках РФ по доходности и условиям пополнения` |
| **Бизнес** | `РКО для ИП с оборотом до 10 млн/год: тарифы, эквайринг, бесплатные операции — Сбер, Тинькофф, Точка, Модульбанк` |
| **Социальные** | `Карта ветерана СВО: условия, привилегии, льготы в банках-участниках проекта (Сбер, ВТБ, ПСБ, Газпромбанк)` |
| **Регуляторика** | `Каков средний CIR (cost-to-income) топ-10 банков за 2025 год по отчётности ЦБ` |

### Отзывы и качество

- `Топ-10 жалоб клиентов на Тинькофф за последний месяц`
- `Что чаще всего хвалят клиенты Сбера в отзывах за квартал`
- `Сравни тональность отзывов на мобильные приложения Сбер/ВТБ/Альфа`

### Маркет-офферы (структурированные данные)

- `Покажи актуальные ставки по вкладам от 1 млн руб на 12 мес` *(тянет из v_offer_current)*
- `Лучшие предложения по кредитным картам с грейс-периодом >100 дней`

## Что в готовом отчёте

После deep-research'а в UI ты видишь:

1. **План исследования** (`OUTLINE`) — какие разделы покрыты
2. **Основной текст** — структурированный markdown с цитатами `[1] [2] [3]`
3. **Карточки источников** — кликабельные, открываются в новой вкладке
   - 🟢 high trust (≥0.85) — регуляторы / IR
   - 🟡 mid trust (0.55-0.85) — официальные сайты банков, агрегаторы
   - 🔴 low trust (<0.55) — блоги/форумы (часто исключаются)
4. **Графики** (если есть числовые сравнения) — Chart.js, можно скачать
5. **Verification report** — какие факты не прошли проверку
6. **Кнопка `Download PDF`** — экспорт в полноценный документ A4

## CLI — без UI

Для интеграции в скрипты / cron:

```bash
source .venv/bin/activate

# Ингест документа в БД
python -m bank_audit.cli ingest-url https://www.cbr.ru/statistics/bank_sector/

# Запуск seed-скрипта (демо-данные)
python scripts/demo_seed.py

# Принудительная переиндексация банка
python -m bank_audit.cli reindex --bank sberbank
```

Полный список:
```bash
python -m bank_audit.cli --help
```

## API — программный доступ

Сервер открывает HTTP API:

| Endpoint | Метод | Что делает |
|---|---|---|
| `/api/ai/analyze` | POST (SSE) | Стриминг deep-research отчёта |
| `/api/rag/ingest-url` | POST | Ингест произвольного URL в БД |
| `/api/rag/search` | POST | Семантический поиск |
| `/api/ai/export-pdf` | POST | Сгенерировать PDF из markdown+sources |
| `/api/banks` | GET | Список банков в БД |
| `/api/offers` | GET | Маркет-офферы |
| `/api/reviews` | GET | Отзывы клиентов |

Пример:
```bash
curl -N -X POST http://127.0.0.1:8000/api/ai/analyze \
  -H "Content-Type: application/json" \
  -d '{"question":"Сравни вклады Сбера и ВТБ","deep":true}'
```

Полная Swagger-документация: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)

## Советы

1. **Чем конкретнее вопрос — тем лучше отчёт.** «Сравни банки» — слабо. «Семейная ипотека в Сбер/ВТБ/Альфа: ставка, ПВ, требования к доходу» — отлично.
2. **Перечисляй банки явно** если хочешь конкретное сравнение. Иначе resolver сам выберет топ-5 по покрытию в БД.
3. **Социальные продукты** (карта ветерана, маткапитал, военная ипотека) — система автоматически подтянет ЦБ РФ / pravo.gov.ru / mil.ru / gosuslugi.ru.
4. **Если отчёт получился короткий** — нажми «🔄 Уточнить» (запустит agent-loop +1 итерация).
5. **PDF лучше выглядит на A4 ландшафте** для отчётов с широкими таблицами.

## Что НЕ умеет (пока)

- ❌ Загружать собственные документы в БД из UI (только через CLI / API)
- ❌ Сравнивать банки за разные даты (snapshot-based diff)
- ❌ Выгружать ответы в Excel — только PDF и копия markdown
- ❌ Авторизация пользователей (single-user mode)
- ❌ Real-time мониторинг изменения тарифов (планируется)
